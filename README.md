# TerraLab — Trainee Data Analytics

Repositório do programa de trainee em Data Analytics do TerraLab.

## Estrutura

- `sprint1/` — Fundamentos e ambiente com Docker (Airflow + PostgreSQL)

## Como rodar a Sprint 1

```bash
cd sprint1
docker compose up -d
```

Acesse o Airflow em: http://localhost:8080
